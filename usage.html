What is this?

This represents the number of current installs that were running at midday local time (to the install) and the number of players that were connected at that exact moment. There is no error checking or recovery, and if YAMS isn't running at midday, nothing is logged for that install that day. Therefore the number of installs is likely higher, this is just for fun really!
