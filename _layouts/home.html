Yet Another Minecraft Server

A simple way to run your own Minecraft server on Windows.

Download

YAMS is not quite cooked yet, but when it is a download link will appear here.

Why use YAMS?

"Always on" server applications in Windows need to be installed and run as a Windows Service. The standard Minecraft server doesn't offer this as an option, so YAMS sits on top of the vanilla (standard) Minecraft server, and keeps it running all the time.

For more details on what YAMS does and doesn't do, see the about page

Open Source

Download the code and customise YAMS for your own needs.

You can also clone the project with Git by running: 

$ git clone git://github.com/richardbenson/YAMS
